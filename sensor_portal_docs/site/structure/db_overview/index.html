Core database Tables

archiving_archive

Stores external archive configuration like credentials and address.

archiving_tarfile

Represents a compressed archive file, including storage status and path.

auth_group, auth_permission, auth_group_permissions

Standard Django auth system tables for group and permission management.

user_management_user

Stores user accounts and metadata such as organization and bio.

data_models_datafile

Stores metadata about uploaded files including: - file_name, file_format, recording_dt, path - JSON fields: extra_data, linked_files - Flags: local_storage, archived, do_not_remove

data_models_deployment

Represents field deployments of devices with: - Spatial info (latitude, longitude, point) - Timestamps for deployment_start, deployment_end

data_models_device

Represents devices in the system, includes: - device_ID, name, model_id, update_time, autoupdate

data_models_project

Defines research projects, including: - project_ID, name, principal_investigator, organisation - Links to archives and sites

observation_editor_observation

Stores labeled observations including: - label, obs_dt, bounding_box, taxon_id, confidence

observation_editor_taxon

Defines taxonomy used in observations: - species_name, species_common_name, taxonomic_level

Relationships and Linking Tables

  • data_models_device_annotators, managers, viewers: Associate users with devices.
  • data_models_deployment_annotators, managers, viewers: Associate users with deployments.
  • data_models_project_annotators, managers, viewers: User-project associations.
  • data_models_deployment_project: Many-to-many between deployments and projects.
  • observation_editor_observation_data_files: Links observations to data files.

Task and Job Scheduling Tables

  • django_celery_beat_*: Celery periodic task schedules (crontab, intervals, solars)
  • django_celery_results_*: Celery result backends
  • data_models_projectjob: Project-level Celery jobs with arguments
  • data_models_project_automated_tasks: Links projects to jobs

Token and Session Management

  • authtoken_token: API token authentication
  • token_blacklist_*: JWT token blacklisting
  • django_session: Django session storage

Miscellaneous

  • external_storage_import_datastorageinput: Config for external storage systems
  • data_packages_datapackage: Bundles of data files with metadata
